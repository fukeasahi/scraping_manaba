LINE NotifyのAPI Tokenの発行方法

1.LINE Notifyのページにアクセス

LINEアプリのアカウントを持っている人はLINE Notifyを使用することができます。 まず、LINE Notifyのページにアクセスしてください LINE Notifyのページはこちら。 アクセス後、右上の「ログイン」のボタンをクリックしてください。

2.LINE Notifyにログイン

上の写真のようなログイン画面が表示されたら、LINEに登録しているメールアドレスとパスワードでログインします。 登録メールアドレスは、LINEアプリの『設定>アカウント』から確認することが出来ます。

3.マイページにアクセス

ログイン後、画面右上のボタンをクリックし、ポップアップされた画面のうち、「マイページ」をクリックしてください。

4.「トークンを発行する」をクリック

写真のような「トークンを発行する」というボタンがあるので、クリックしてください。

6.トークンの発行

「トークン名」を記入し、通知を送信するトークルームを選択して下さい。(他人が閲覧できるトークルームを選択しないように気をつけて下さい)

7.発行したトークンのコピー

発行したトークンをコピーして下さい。

8.MLIの会員登録でトークンの貼り付け、登録

コピーしたトークンをMLIの会員登録画面で、「LINE NotifyのAPI Token」、「LINE NotifyのAPI Token(確認)」に貼り付けて、他の情報と一緒に登録して下さい。
